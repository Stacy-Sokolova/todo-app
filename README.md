# Todo App

REST API для управления задачами

# Для запуска приложения

- определить переменные окружения для запуска сервера и базы данных
- перед запуском приложения впервые необходимо применить миграции к базе данных
```
make migrate
```
- для запуска приложения
```
make run
```
